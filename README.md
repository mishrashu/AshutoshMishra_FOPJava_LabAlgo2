# AshutoshMishra_FOPJava_LabAlgo2
Lab2AlgorithmAssignment

This assignment solves 2 problems

1. PayMoney. processes thousands of transactions daily amounting to crores of Rupees.
They also have a daily target that they must achieve. Given a list of transactions done by
PayMoney and a daily target, your task is to determine at which transaction PayMoney
achieves the same. If the target is not achievable, then display the target is not achieved.

2. You are a traveler and traveling to a country where the currency denominations are
unknown and as you travel you get to know about the denomination in random order.
You want to make a payment of amount x, in such a way that the higher denomination is
used to make exact payment.

Sorting of denominations uses merge sort
